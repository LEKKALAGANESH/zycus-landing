# Zycus hero mockups

SVG wireframe mockups of the Zycus hero section across three viewports (desktop 1200 × 600, tablet 768 × 600, mobile 375 × 600). These are low-fidelity layouts for the client evaluator to visualise the design before the live demo URL is available. Production pixel-accurate screenshots should replace these files once the InstaWP / InfinityFree sandbox is live.
